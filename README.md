# CMM解释器

###武汉大学2017级解释器构造实践
###小组成员：张雷、汪志豪、王亮、杨天舒、文萱、刘佳媚
#
#####本项目已编译为EXE文件Interpreter.exe
#####项目运行环境：Java 1.8
#
#####运行Main文件可打开可视化界面
#####运行MainTest文件可在控制台中运行CMM语言示例代码
#####运行Test文件可执行单元测试
#
#####Antlr4语法文件及其生成文件于CMM文件夹中
#####CMM语言代码示例于DemonstrationProgram文件夹中
#####可视化界面代码于Interface文件夹中
#####EXE编译文件于META-INF中
#####语法树遍历、语义分析及解释执行代码于SemanticAnalysis文件夹中
#####测试代码于Test文件夹中
